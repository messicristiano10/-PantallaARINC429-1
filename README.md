# PantallaARINC429

Proyecto web para seleccionar grupos de instrumentos aeronáuticos, elegir un instrumento asociado y agregar valores.

## Cómo usar
1. Abrir `index.html` en el navegador.
2. Seleccionar un grupo (Motores, Control, Navegación).
3. Seleccionar un instrumento.
4. Agregar valores al instrumento seleccionado.
